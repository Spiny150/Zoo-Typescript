Bienvenue sur l'application de gestion du Zoo !

Ce projet est une application de gestion pour un zoo, développée avec Angular pour le frontend et NestJS pour le backend. Elle permet de gérer les animaux et leurs enclos, d'ajouter de nouveaux pensionnaires, de les soigner ou de les relâcher. L'objectif est de fournir une interface simple et intuitive pour les administrateurs du zoo.

Explorez les différentes sections pour voir la liste des animaux, gérer les enclos, et bien plus encore.
